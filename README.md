# Proyecto Web - Somos Empleables

Este proyecto utiliza **Astro** y **Tailwind CSS**.

## Estructura del Proyecto

*   `src/components`: Componentes reutilizables.
*   `src/layouts`: Plantillas principales (Layout.astro).
*   `src/pages`: Rutas y páginas del sitio.
*   `src/styles`: Archivos CSS globales (global.css).

## Comandos Disponibles

Todos los comandos se ejecutan desde la raíz del proyecto, en una terminal:

| Comando                   | Acción                                           |
| :------------------------ | :----------------------------------------------- |
| `npm run dev`             | Inicia el servidor de desarrollo local.          |
| `npm run build`           | Compila el sitio para producción.                |
| `npm run preview`         | Previsualiza la versión compilada localmente.    |

## Tecnologías

*   [Astro](https://astro.build/) - Framework web.
*   [Tailwind CSS](https://tailwindcss.com/) - Estilos utilitarios.
